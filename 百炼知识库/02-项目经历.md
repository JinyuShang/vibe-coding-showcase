## 项目1：家庭食材管理系统

### 项目简介
管理家庭食材的库存与保质期，减少浪费，让厨房更井井有条。

### 项目详情
一款帮助家庭管理食材库存的Web应用。通过记录食材的购买日期、保质期等信息，智能提醒即将过期的食材，减少食物浪费，让家庭厨房管理更加高效有序。

### 核心功能
- 食材分类管理
- 保质期预警
- 消费记录追踪
- 库存管理

### 技术栈
- **开发工具**：Enter
- **数据库**：Supabase
- **部署平台**：Vercel

### 项目亮点
- 完整的CRUD功能实现
- 数据库设计与关联
- 实时数据更新
- 用户友好的界面设计

### 在线演示
http://www.familypantry.de5.net

## 项目4：三维模型标注平台

### 项目简介
在线三维模型标注工具，支持多角度查看与精准标注。

### 项目详情
在线三维模型标注工具。支持在3D空间中对模型进行精准标注，可多角度旋转查看模型细节。适用于3D模型审查、设计反馈标注等场景，为三维协作提供便捷的在线解决方案。

### 核心功能
- 3D模型展示
- 多角度旋转查看
- 空间标注功能
- 标注记录保存

### 技术栈
- **开发工具**：Cursor、Claude Code
- **3D渲染**：Three.js
- **前端框架**：Next.js

### 项目亮点
- 3D交互实现
- 空间标注技术
- 专业应用场景
- 复杂交互逻辑

### 项目类型
本地小工具（无在线演示）

## 项目技术总结

### 掌握的技术
- **前端框架**：Next.js 16 (App Router)、React
- **编程语言**：TypeScript
- **样式方案**：Tailwind CSS
- **数据库**：Supabase
- **3D技术**：Three.js
- **AI工具**：Cursor、Claude Code、百炼智能体

- ✅ 从简单到复杂的学习路径
- ✅ 涵盖多种技术栈
- ✅ 注重实用性
