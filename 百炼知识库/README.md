# 百炼智能体知识库使用指南

## 📚 知识库文档清单

本文件夹包含5个Markdown文档，用于配置百炼智能体的知识库：

1. **01-个人介绍.md** - 尚瑾瑜的个人背景、学习经历和联系方式
2. **02-项目经历.md** - 4个实战项目的详细信息
3. **03-技能栈.md** - 7大核心技能详解
4. **04-学习历程.md** - 8个课程的完整学习路径
5. **05-技术知识.md** - 前端开发、AI开发等技术知识

---

## 🚀 上传到百炼平台的步骤

### 步骤1：登录阿里云百炼平台
访问：https://bailian.console.aliyun.com/

### 步骤2：创建知识库
1. 点击左侧菜单的"知识库"
2. 点击"创建知识库"
3. 输入知识库名称，例如："尚瑾瑜AI助手知识库"
4. 选择知识库类型："文档型"
5. 点击"确定"

### 步骤3：上传文档
1. 进入新创建的知识库
2. 点击"上传文档"
3. 选择以下5个文件：
   - ✅ 01-个人介绍.md
   - ✅ 02-项目经历.md
   - ✅ 03-技能栈.md
   - ✅ 04-学习历程.md
   - ✅ 05-技术知识.md
4. 点击"开始上传"
5. 等待文档解析完成

### 步骤4：创建智能体
1. 点击左侧菜单的"智能体"或"Agent"
2. 点击"创建智能体"
3. 填写基本信息：
   - **名称**：尚瑾瑜AI助手
   - **描述**：帮助访客了解尚瑾瑜的项目、技能和学习历程

### 步骤5：配置智能体
1. **人设/角色**（粘贴以下内容）：
```
你是尚瑾瑜的AI助手，帮助访客了解他的项目经历、技能栈、学习历程和技术能力。

你的特点：
- 友好、专业、热情
- 基于知识库内容回答问题
- 回答简洁明了，重点突出
- 适当使用emoji增加亲和力

回答问题时：
1. 优先使用知识库中的信息
2. 如果知识库中没有相关信息，诚实告知并建议访问网站或联系尚瑾瑜
3. 保持积极正面的态度
4. 可以主动推荐查看项目、技能等页面
```

2. **欢迎语**（粘贴以下内容）：
```
你好！👋 我是尚瑾瑜的AI助手。我可以帮你了解他的项目经历、技能栈、学习历程，或者回答技术问题。有什么想问的吗？
```

3. **关联知识库**：
   - 在"知识库"设置中，选择刚才创建的知识库
   - 设置检索模式为"向量检索"

### 步骤6：测试智能体
1. 在右侧的测试对话框中输入问题
2. 查看AI的回答是否准确
3. 根据需要调整人设和知识库

### 步骤7：发布和获取API信息
1. 测试满意后，点击"发布"
2. 发布后，进入"API管理"或"调用信息"
3. 记录以下信息（用于接入网站）：
   - **App ID**（应用ID）
   - **API Key**（或Access Key）
   - **Endpoint**（API调用地址）

## ⚠️ 注意事项

### 文档格式
- 所有文档都是Markdown格式（.md）
- 百炼平台支持Markdown文件解析
- 文档编码为UTF-8

### 内容更新
如果需要更新知识库内容：
1. 修改对应的.md文件
2. 在百炼平台重新上传文档
3. 或使用"增量更新"功能

### 知识库优化
- 定期查看智能体的问答日志
- 根据用户问题补充知识库内容
- 调整检索参数提升准确度
