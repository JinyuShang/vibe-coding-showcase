# 尚瑾瑜的技能栈详解

## 技能概览

通过Vibe Coding实战训练营的系统学习，尚瑾瑜掌握了7大核心技能，形成了完整的产品开发能力体系。

---

## 技能1：两大思维

### 技能描述
**产品思维 + 工程思维**的融合运用。

### 详细说明
- **产品思维**：从用户需求出发，设计有价值的产品功能
- **工程思维**：考虑技术可行性、可维护性和扩展性
- **融合应用**：在产品设计和开发中平衡用户体验和技术实现

### 实践应用
在所有项目中，尚瑾瑜都能从产品角度思考需求，从工程角度实现功能，确保产品既好用又可靠。

---

## 技能2：产品的构成

### 技能描述
理解产品由**触达、分发、形态、服务、数据**五个维度组成。

### 五维度详解
- **触达**：用户如何发现和接触产品（SEO、分享、推广）
- **分发**：产品如何交付到用户手中（应用商店、Web、下载）
- **形态**：产品的具体形式（Web应用、移动应用、桌面应用）
- **服务**：产品提供的核心价值和功能
- **数据**：用户数据、产品数据的收集、分析和应用

### 实践应用
在每个项目中，尚瑾瑜都会考虑这五个维度，确保产品的完整性和可持续性。

## 技能4：编写PRD文档

### 技能描述
系统性梳理需求并撰写PRD（产品需求文档）的技能。

### PRD文档要素
- 需求背景和目标
- 用户画像和场景
- 功能详细说明
- 交互流程图
- 验收标准
- 技术要求

### 实践应用
在开发每个项目前，尚瑾瑜都会编写详细的PRD文档，确保需求清晰、目标明确。

## 技能6：工具清单

### 技能描述
熟练使用多种AI开发工具的能力。

### 核心工具

#### 💡 Enter
- **定位**：云端自动化开发平台
- **用途**：快速原型开发、自动化流程
- **掌握程度**：熟练使用

#### ⚡ Cursor
- **定位**：AI代码编辑器（AI IDE）
- **用途**：日常代码编写、AI辅助编程
- **掌握程度**：主要开发工具

#### 🔧 Claude Code
- **定位**：AI命令行工具（AI CLI）
- **用途**：项目开发、代码重构、技术咨询
- **掌握程度**：深度使用，掌握自定义技能

#### 📦 GitHub
- **定位**：版本管理和代码托管平台
- **用途**：代码版本控制、协作开发
- **掌握程度**：熟练使用Git工作流

#### 🗄️ Supabase
- **定位**：开源Firebase替代方案
- **用途**：数据库、认证、存储
- **掌握程度**：能独立构建后端服务

#### 🚀 Vercel
- **定位**：前端部署平台
- **用途**：Next.js应用部署、自动化部署
- **掌握程度**：熟练部署和监控

#### 📊 PostHog
- **定位**：产品分析和数据埋点工具
- **用途**：用户行为分析、数据追踪
- **掌握程度**：基础使用
